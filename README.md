# Tarjeta de Visita Digital Corporativa - Carmen Martín Oliveira 💼

Este repositorio aloja la tarjeta de visita digital e interactiva de **Carmen Martín Oliveira**, Gerente Adjunto en el Centro Comercial Holea (Huelva). 

El proyecto está diseñado con un enfoque corporativo de alto nivel, optimizado para dispositivos móviles y optimizado para redes profesionales.

## 🚀 Características del Proyecto
* **Diseño Premium:** Paleta de colores ejecutiva basada en azul medianoche y oro satinado corporativo.
* **Acceso Inmediato:** Enlaces directos y limpios a su perfil oficial de LinkedIn y correo electrónico profesional.
* **Hosting Cloud:** Desplegado de forma eficiente y gratuita a través de **GitHub Pages**.

## 🛠️ Tecnologías y Herramientas
* **Frontend:** HTML5 y CSS3 (Diseño responsivo).
* **Control de Versiones & Hosting:** Git, GitHub y GitHub Pages.
